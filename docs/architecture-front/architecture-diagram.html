OpenCamille Agent Harness Architecture

v0.1 proves the harness: CLI, runtime control, provider streaming, tools, approvals, memory, skills, MCP, recorder, resume.

State authority

  • • Runtime Control is the only live ConversationHistory writer
  • • AgentLoop and Tools return results, not session mutations
  • • Resume reads checkpoint.json, not event replay

v0.1 proof slice

  • • CLI, provider streaming, tools, approvals, memory compaction
  • • Skills and stdio MCP tools use the same Tools path
  • • Tool output is normalized, truncated, and redacted

Deliberate exclusions

  • • No ToolExecutor, StateMachineEngine, FileSystem, Shell class
  • • No Web/API/IDE, subagents, RAG, plugin package format
  • • No MCP HTTP/auth/resources/prompts in v0.1
